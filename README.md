# 项目简介

SpringBoot3+Vue农产品推荐系统是一款基于SpringBoot3和Vue3的农产品销售管理系统，旨在帮助农产品销售者实现商品的线上销售和推荐。

## 核心功能
- **协同过滤推荐算法**：基于用户行为和商品信息，实现精准的商品推荐。
- **支付宝沙箱支付**：支持支付宝沙箱支付功能，方便用户进行在线支付。
- **邮箱注册**：用户可以通过QQ邮箱或真实邮箱进行账号注册。
- **角色管理**：
  - 普通用户：商品浏览、购买下单、订单查询、商品评价与收藏、查看个性化推荐。
  - 农户：商品发布与管理、销售数据查看、订单处理。
  - 系统管理员：用户账户管理、商品审核、订单监管、数据统计与分析。
- **前端功能模块**：
  - 首页门户：轮播广告、推荐商品、分类导航。
  - 商品中心：商品列表/详情、关键词搜索。
  - 购物车：商品添加、数量调整、商品移除。
  - 订单流程：下单支付、状态跟踪、物流查询。
  - 用户中心：个人信息管理、收货地址维护、历史订单、我的收藏。
  - 评价系统：发表评价、评价管理。
  - 智能推荐：个性化推荐专区、热销商品展示。
- **后台功能模块**：
  - 用户管理：用户信息审核与权限分配。
  - 商品管理：商品上下架与审核管理。
  - 订单管理：订单处理与发货管理。
  - 评价管理：用户评价审核与数据统计。
  - 数据看板：销售数据与用户行为分析。
  - 系统设置：轮播图配置、商品分类管理。

## 适用场景
- 农产品销售者：通过系统实现线上销售，提高销售效率。
- 农户：发布和管理自己的农产品，查看销售数据，处理订单。
- 系统管理员：管理用户账户、商品、订单，进行数据统计与分析。

## 技术特点
- **开发工具**：IDEA、JDK17、Maven、MySQL5.7/8。
- **技术栈**：SpringBoot3、Mybatis-Plus、Vue3、MySQL。
- **项目文档**：包含2w字文档，方便用户学习和使用。
